# FrontEnd Service

This is a frontend service using ReactJS, NextJS, Tailwind, and Storybook.

## How to install

```bash
npm install
```

## How to run

```bash
npm run dev
```

Open [http://localhost:4000](http://localhost:4000) on your browser.

## Storybook Testing

```bash
npm run storybook
```
